
## Documentation

This is framework for web automation using playwright with JS with POM




## Run Locally

Clone the project

```bash
  git clone https://github.com/Mukesh-50/PlaywrightFrameworkWithPOM.git
```

Go to the project directory

```bash
  cd PlaywrightFrameworkWithPOM
```

Install dependencies

```bash
  npm install
```

To run the test

```bash
  npm run test
```

To run the smoke test

```bash
  npm run smoke
```
